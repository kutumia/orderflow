Connect Your Restaurant

Find this in your OrderFlow dashboard → Settings.

Add to Your Theme

Two ways to add ordering to your Shopify store:

1
OrderFlow Menu Block (Recommended) Go to your theme editor → Add section → "OrderFlow Menu". Embeds your full ordering menu inline on any page.
2
OrderFlow Button Block Add section → "OrderFlow Button". Adds a floating "Order Online" button that opens an ordering popup on every page.

Manual embed: If you prefer, paste this code into any Liquid template:

<script src="https://orderflow.co.uk/widget.js" data-restaurant="your-slug"></script>

Order Sync

OrderFlow orders are automatically created as draft orders in your Shopify admin for accounting and reconciliation.
