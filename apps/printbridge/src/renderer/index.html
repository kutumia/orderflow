OrderFlow
Kitchen Printer Setup

Welcome!

This app connects your kitchen printer to OrderFlow so orders print automatically. Setup takes about 60 seconds.

You'll need:

• Your OrderFlow dashboard login

• A USB thermal receipt printer (80mm or 58mm)

• The printer API key from your dashboard

Connect to OrderFlow

Enter your server URL and printer API key from Dashboard → Printer.

Find this in Dashboard → Printer → Step 1

Select Printer

Choose which printer to use for order receipts.

Helps identify this device in the dashboard

✅

All Set!

OrderFlow Printer is ready. It will run silently in your system tray.

What happens now:

• Orders will print automatically when customers pay

• Look for the printer icon in your system tray

• The app starts automatically when your PC boots

• Updates install automatically in the background
