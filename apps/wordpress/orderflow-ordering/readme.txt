Add online ordering to your restaurant website. Customers order and pay directly — no marketplace commission.

== Description ==

OrderFlow lets restaurants accept orders through their own website instead of paying 25-35% commission to Just Eat, Deliveroo, or Uber Eats.

**Features:**

* Floating "Order Online" button on every page
* Embed your full menu inline with the `[orderflow_menu]` shortcode
* Gutenberg block for easy page building
* Mobile-optimised ordering experience
* Card payments via Stripe
* Orders print automatically in your kitchen

**How It Works:**

1. Sign up at [orderflow.co.uk](https://orderflow.co.uk)
2. Set up your menu and connect payments
3. Install this plugin and enter your restaurant slug
4. Customers can now order from your website

**Requirements:**

* An active OrderFlow account ([sign up for free trial](https://orderflow.co.uk/register))

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/orderflow-ordering/`
2. Activate the plugin through the 'Plugins' menu
3. Go to Settings → OrderFlow
4. Enter your restaurant slug (found in your OrderFlow dashboard)
5. The ordering button will appear on your website automatically

== Frequently Asked Questions ==

= Do I need an OrderFlow account? =

Yes. Sign up for a 14-day free trial at orderflow.co.uk.

= How much does it cost? =

The plugin is free. OrderFlow plans start from £49/month with no per-order commission.

= Can I customise the button? =

Yes. Change the button text, colour, and position in Settings → OrderFlow.
